# Personal Portfolio Website Setup Guide

## Quick & Professional Portfolio on Notion or Carrd

---

## Platform Comparison

| Feature | Notion | Carrd |
|---------|--------|-------|
| **Setup Time** | 30-60 minutes | 1-2 hours |
| **Learning Curve** | Easy (if familiar) | Moderate |
| **Customization** | Medium | High |
| **Free Tier** | Yes, generous | Yes, limited |
| **Best For** | Quick setup, content-heavy | Visual appeal, branding |
| **SEO** | Poor | Better |
| **Mobile Responsive** | Yes | Yes |

---

## OPTION 1: NOTION PORTFOLIO (Fastest Setup)

### Step 1: Create a New Page

1. Open Notion
2. Click **+ New page** in sidebar
3. Add cover image (tech-themed or clean gradient)
4. Add icon (💼, 📝, or 👨‍💻)
5. Title: "[Your Name] — AI/ML Technical Writer"

### Step 2: Page Structure

Use this exact structure with these sections:

### Step 4: Make Page Public

1. Click **Share** button (top right)
2. Scroll to **Publishing to web**
3. Toggle **On**
4. Copy the published URL
5. Customize URL if desired (click edit icon next to URL)

Your portfolio will be at:
```
https://notion.site/YOUR_PAGE_NAME
```

### Step 5: Notion Styling Tips

- Use **Callout blocks** (⌘/ then type "callout") for emphasis
- Use **Toggle headers** for collapsible sections
- Add **dividers** (---) between major sections
- Use **columns** for side-by-side content
- Embed **GitHub files** directly using `/embed`

## OPTION 2: CARRD PORTFOLIO (More Visual)

### Step 1: Create Account & New Site

1. Go to **carrd.co** and sign up (free tier available)
2. Click **Create New** → Choose **Blank** or **Portfolio template**
3. Name your site: "[Your Name] — AI/ML Technical Writer"

### Step 2: Configure Site Settings

Click **Settings** (gear icon):

```
Site Name: [Your Name]
Title: [Your Name] — AI/ML Technical Writer
Description: AI/ML Technical Writer specializing in API documentation, tutorials, and technical content for developer audiences.
Favicon: Upload small logo or use text icon
```

### Step 3: Build Sections (Add Blocks)

Click **+** to add blocks. Use this structure:

#### Block 1: Header
- Type: **Header**
- Content: Your name + title
- Style: Centered, large font
- Background: Gradient or solid color

```
[Your Name]
AI/ML Technical Writer
Transforming Complex LLM Concepts into Clear Guides
```

#### Block 2: Text (About)
- Type: **Text**
- Content: Brief intro paragraph
- Style: Centered, readable font size

#### Block 3: Buttons (Portfolio Links)
- Type: **Buttons**
- Layout: Grid or stacked
- Buttons:
  - "📄 API Documentation Sample"
  - "📚 RAG Tutorial Series"
  - "🧠 How Attention Works"
  - "💼 LinkedIn Profile"

#### Block 4: Divider
- Type: **Divider**
- Style: Simple line

#### Block 5: Text (Skills)
- Type: **Text** or **Features**
- Content: Skills list with icons

```
📝 API Documentation
📚 Technical Tutorials  
🧠 LLM Architecture
💻 Python & LangChain
🔍 RAG Systems
```

#### Block 6: Social Icons
- Type: **Social**
- Add: LinkedIn, GitHub, Email icons
- Link to your profiles

### Step 4: Carrd Styling Tips

**Colors:** Use professional palette
```css
Primary: #1a1a2e (dark blue)
Secondary: #16213e (navy)
Accent: #e94560 (coral/red)
Text: #ffffff (white)
Background: #0f0f23 (very dark)
```

**Fonts:**
- Headings: Montserrat or Poppins
- Body: Open Sans or Roboto

### Step 5: Publish

1. Click **Publish** button
2. Choose subdomain: `yourname.carrd.co`
3. Or connect custom domain (paid feature)
4. Click **Confirm**

Your portfolio will be at:
```
https://yourname.carrd.co
```

## COMPARISON: Sample Portfolio Content

### Notion Advantages:
- ✅ Faster to set up
- ✅ Easier to update content
- ✅ Better for long-form content
- ✅ Free tier very generous
- ❌ Less visual customization
- ❌ Poor SEO

### Carrd Advantages:
- ✅ More visual appeal
- ✅ Better branding control
- ✅ Better SEO than Notion
- ✅ Looks more "professional website"
- ❌ Slightly steeper learning curve
- ❌ Free tier has limitations

---

## RECOMMENDATION

**For speed and simplicity:** Use **Notion** — you can have a portfolio live in 30 minutes.

**For visual appeal and branding:** Use **Carrd** — worth the extra hour for a more polished look.

**Hybrid approach (recommended):**
1. Start with Notion for speed
2. Migrate to Carrd later when you have time
3. Or use both: Notion for content, Carrd as landing page linking to Notion
